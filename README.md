SQL Learning from [CampusX](https://www.youtube.com/@campusx-official) 🚀

This repository contains my structured SQL learning journey from CampusX, including practice queries and real-world use cases.

---

## 📂 Topics Covered

* **01_Basics** → Basic queries, filtering, sorting
* **02_DDL_commands** → CREATE, DROP, TRUNCATE, ALTER
* **03_DML_commands** → INSERT, SELECT, UPDATE, DELETE
* **04_Joins** → CROSS JOIN, INNER JOIN, LEFT JOIN, RIGHT JOIN, SELF JOIN
* **05_Aggregations** → ORDER BY, GROUP BY, HAVING
* **06_Subqueries** → Nested queries and filtering
* **07_Window_Function** → Ranking, cumulative sum, rolling average, lead/lag
