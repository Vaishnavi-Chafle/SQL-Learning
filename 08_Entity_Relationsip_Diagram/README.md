# 📊 Entity Relationship Diagram (ERD)

## 📌 Overview
This directory contains the Entity Relationship Diagram for the Swiggy database project.

## 🧠 What is ER Diagram?
An ER diagram represents the structure of a database, showing entities, attributes, and relationships between them.
